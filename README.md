# ROS self-learning Repository
ROS beginner_tutorials

## How to build:
Remove old complied files (if exist) and rebuild
``` bash
    # make sure in the same levl of src directory
    rm -r devel/ build/
    catkin_make
    # run setup script
    source devel/setup.bash
```
